format为取值类型，有以下类型：

  • string 字符串
  • color 颜色值，如#ffffff
  • dimension 尺寸，xml中设置为dp/dip, 字体为sp
  • integer 数值 1
  • enum 枚举类型
  • reference 引用，如@drawable/ic_launcher
  • float 浮点类型 如1.0
  • boolean 布尔类型 true或false
  • fraction 百分数 如100%
  • flag 位或运算
